# SELLM 用户使用手册

> 面向特殊教育(自闭症谱系等 8 类障碍)的评估与 IEP 生成系统。
> 按角色组织:超级管理员 / 机构管理者 / 教师(管理端 Web),家长(微信小程序)。

## 一、角色与权限

| 角色 | 入口 | 能做什么 |
|------|------|---------|
| **SUPER_ADMIN 超级管理员** | 管理端 Web | 跨机构管理:机构、所有用户、量表库 |
| **MANAGER 机构管理者** | 管理端 Web | 本机构:班级、教师审核、**微信家长审核激活**(分配机构+绑孩子) |
| **TEACHER 教师** | 管理端 Web | 本机构孩子:建档、评估、报告、IEP、教学/科研/教具 |
| **PARENT 家长** | 微信小程序 | 仅自己孩子:查看报告、生成家庭 IEP、教具推荐、成长记录 |

> **两道权限**:登录态(JWT)+ 行级(SUPER_ADMIN 跨机构 / MANAGER·TEACHER 限本机构 / PARENT 限自己孩子)。

## 二、八类障碍类型

孤独症(ASD)、发育迟缓、智力障碍、语言障碍、感统失调、脑瘫、注意缺陷多动障碍(ADHD)、听视障。

## 三、AI 使用须知(重要)

- **AI 只产草案**:评估报告、IEP、家庭 IEP、教案等由 AI 生成草稿(DRAFT),**须教师/家长人工编辑后「定稿」(FINALIZED)**。
- **定稿后冻结**:已定稿内容不可再改,PDF 仅对已定稿内容开放下载。
- **多模态识别只产建议**:图片/笔记识别给出指标评分建议,教师确认后才计入正式评估。

## 四、登录

- **管理端(Web)**:浏览器打开管理端地址 → 账号密码登录。
- **小程序(家长)**:微信打开小程序 → 微信一键登录(首次登录后状态为「待审核」,需机构管理者激活)或账号密码登录。

## 五、超级管理员 / 机构管理者(管理端)

### 机构与班级
- 机构管理(超管):新建机构、指定机构管理者。
- 班级管理(管理者):本机构建班、分配教师。

### 用户审核
- **教师/家长注册审核**:审核注册申请,通过后激活。
- **微信家长激活(管理者)**:小程序微信登录的家长进入「微信家长待激活」列表 → 管理者填孩子姓名/障碍类型/班级 → 激活(分配本机构 + 建孩子档案 + 账号转 ACTIVE)。

### 量表库(超管)
- 维护评估量表(指标、维度、计分区间)。

## 六、教师(管理端)核心流程

```
建孩子档案 → 选量表评估打分 → 生成评估报告(AI草案)→ 编辑定稿 → 下载报告PDF
                                              ↓
                          基于定稿报告生成 IEP(AI草案)→ 编辑定稿 → 下载IEP PDF
                                              ↓
                          教学:据定稿IEP生成教案/课件草案 → 定稿
```

1. **建档**:孩子模块 → 新建(姓名、障碍类型、班级);姓名等 PII 加密落库。
2. **评估**:选孩子 + 量表 → 逐指标打分提交。可选上传图片/训练笔记由多模态给评分建议(教师确认后计入)。
3. **报告**:据评估生成报告草案 → 编辑 → 定稿 → PDF。
4. **IEP**:**报告定稿后**据结论生成 IEP 草案 → 编辑 → 定稿 → PDF。
5. **教学/科研/教具**:生成教案课件、信效度分析/课题书、教具推荐与文生素材(均 AI 草案,人工定稿)。

## 七、家长(微信小程序)

底部三标签:**孩子 / 教具 / 我的**。

| 页面 | 操作 |
|------|------|
| 孩子列表 | 看自己孩子卡片;顶部复评/IEP 到期提醒 |
| 孩子详情 | 月目标、基线、干预进度 |
| 家庭 IEP | 列表生成家庭训练计划草案(**需该孩子已有定稿评估报告**);定稿后下载 PDF |
| 报告 | 只读查看,定稿报告可下载 PDF(家长不可生成报告) |
| 教具推荐 | 按障碍类型下拉,看推荐教具卡片 |
| 文生素材 | 输入描述提交 → 轮询生成 → 查看插图(点列表项展开预览,可全屏) |
| 成长记录 | 孩子日常记录 |
| 我的 | 登出 |

> 家庭 IEP 报「请等待评估报告定稿」= 该孩子还没有教师定稿的评估报告,需先由教师完成评估并定稿。

## 八、常见问题

- **微信登录后进不去?** 首次微信登录状态为「待审核」,需机构管理者在管理端激活(分配机构+绑孩子)。
- **报告/IEP 下载 PDF 提示不可下载?** 仅「已定稿」可下载,先完成编辑并定稿。
- **教具推荐为空?** 该障碍类型暂无匹配教具,或选了「全部」之外的细分类型。
- **文生素材生成较慢?** 真实文生图需 10~30 秒(出现「生成中」属正常),mock 模式即时返回占位图。
- **PDF 中文乱码/缺字?** 服务端未配 `SELLM_PDF_FONT_PATH` 指向 CJK 字体,联系运维。
